## نظام تقسيم المهام

### المبرمج الأول: Frontend Developer
**المسؤوليات:**
- واجهات المستخدم (React/Vue/Angular)
- تصميم المكونات
- التفاعل مع المستخدم
- الاستجابة للشاشات المختلفة

**اللغات المطلوبة:**
- HTML/CSS/JavaScript
- React أو Vue أو Angular
- TypeScript (اختياري)

### المبرمج الثاني: Backend Developer
**المسؤوليات:**
- APIs وقواعد البيانات
- المنطق الخلفي للتطبيق
- الأمان والمصادقة
- إدارة الخوادم

**اللغات المطلوبة:**
- Node.js أو Python أو PHP
- SQL لقواعد البيانات
- REST APIs

### المبرمج الثالث (أنت): Full Stack + Project Manager
**المسؤوليات:**
- ربط Frontend مع Backend
- مراجعة كود الفريق
- إدارة المشروع والمهام
- حل المشاكل المعقدة

## قواعد استخدام Git

```bash
# كل مبرمج يعمل على branch منفصل
git checkout -b feature/اسم-المهمة

# قبل الدمج، مراجعة الكود
git add .
git commit -m "وصف واضح للتغييرات"
git push origin feature/اسم-المهمة

# طلب مراجعة قبل الدمج
```
