### المبرمج الثاني (Backend Developer)
**المسؤوليات:**
- إدارة قواعد البيانات المتعددة
- تطوير APIs التي تدعم Multi-Tenant
- ضمان الأمان والعزل بين العملاء
- مراقبة الأداء

**المهام اليومية:**
```javascript
// 1. تطوير APIs آمنة
router.get('/cars', async (req, res) => {
  // التأكد من وجود tenant
  if (!req.tenant) {
    return res.status(400).json({ error: 'Tenant not found' });
  }
  
  // استخدام قاعدة بيانات العميل فقط
  const Car = req.getModel('Car');
  const cars = await Car.find();
  
  res.json({ 
    tenant: req.tenant.name,
    data: cars 
  });
});

// 2. فحص الاتصالات
const checkConnections = async () => {
  for (const tenantId in tenants) {
    try {
      await mongoose.connect(tenants[tenantId].mongoUri);
      console.log(`✅ ${tenantId}: متصل`);
    } catch (error) {
      console.log(`❌ ${tenantId}: خطأ`);
    }
  }
};

// 3. مراقبة الأداء
```

**الأدوات المطلوبة:**
- MongoDB Compass لإدارة قواعد البيانات
- Postman لاختبار APIs
- MongoDB Atlas للمراقبة
