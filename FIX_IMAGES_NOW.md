# إصلاح صور المعرض الكوري - حل فوري

## 🚨 المشكلة
الصور لا تظهر في المعرض الكوري للأدمن والعملاء

## ⚡ الحل السريع (30 ثانية)

### الخطوة 1: تشغيل سكريبت الإصلاح
```bash
node scripts/fix-korean-images.js
```

### الخطوة 2: التحقق
افتح المتصفح:
```
https://hmcar.xyz/admin/cars?source=korean_import
```

### الخطوة 3: مسح الكاش
اضغط: `Ctrl + Shift + R` (أو `Cmd + Shift + R` على Mac)

---

## ✅ النتيجة المتوقعة

بعد تشغيل السكريبت:
- ✅ جميع الصور تظهر في لوحة الأدمن
- ✅ الصور تظهر في صفحة المعرض للعملاء
- ✅ الروابط المعطوبة تم إصلاحها
- ✅ السيارات بدون صور حصلت على صور افتراضية

---

## 🔍 ماذا يفعل السكريبت؟

1. يبحث عن جميع السيارات الكورية
2. يفحص روابط الصور
3. يصلح الروابط المعطوبة:
   - روابط تنتهي بـ `_` → يضيف `001.jpg`
   - روابط نسبية → يضيف `https://ci.encar.com`
   - روابط مكررة → يزيل التكرار
   - روابط فارغة → يضيف صورة افتراضية

---

## 📊 مثال على الإصلاح

### قبل:
```
/carpicture/2024/01/15/abc123_
```

### بعد:
```
https://ci.encar.com/carpicture/2024/01/15/abc123_001.jpg
```

---

## 🆘 إذا لم تظهر الصور

### 1. تحقق من تشغيل السكريبت
```bash
node scripts/fix-korean-images.js
```

يجب أن ترى:
```
✅ تم إصلاح روابط الصور بنجاح!
```

### 2. امسح الكاش
- Chrome/Edge: `Ctrl + Shift + R`
- Firefox: `Ctrl + F5`
- Safari: `Cmd + Option + R`

### 3. تحقق من Console
افتح Developer Tools (F12) وتحقق من وجود أخطاء

### 4. تحقق من قاعدة البيانات
```bash
node scripts/check-korean-showroom.js
```
