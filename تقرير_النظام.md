## 🎯 ما يجب عمله لإضافة عميل جديد

### الخطوة 1: إضافة في tenants.json

```json
{
  "newclient": {
    "id": "newclient",
    "name": "اسم المعرض",
    "nameEn": "Showroom Name",
    "description": "وصف المعرض",
    "descriptionEn": "Showroom Description",
    "domains": [
      "newclient.com",
      "www.newclient.com"
    ],
    "mongoUri": "mongodb+srv://user:pass@cluster.mongodb.net/newclient_db",
    "logo": "/uploads/tenants/newclient/logo.png",
    "favicon": "/uploads/tenants/newclient/favicon.ico",
    "theme": {
      "primaryColor": "#3498db",
      "secondaryColor": "#2980b9",
      "accentColor": "#e74c3c",
      "backgroundColor": "#ffffff",
      "textColor": "#2c3e50"
    },
    "contact": {
      "whatsapp": "+966501234567",
      "email": "info@newclient.com",
      "phone": "+966501234567"
    },
    "settings": {
      "currency": "SAR",
      "language": "ar",
      "direction": "rtl"
    },
    "enabled": true
  }
}
```

### الخطوة 2: إنشاء قاعدة بيانات

1. اذهب إلى [MongoDB Atlas](https://cloud.mongodb.com)
2. أنشئ Database جديدة
3. انسخ Connection String
4. ضعه في `mongoUri`

### الخطوة 3: رفع الملفات

```bash
# إنشاء المجلد
mkdir -p uploads/tenants/newclient

# رفع الملفات
# logo.png (400x120px)
# favicon.ico (32x32px)
```

### الخطوة 4: ربط الدومين

#### في Vercel:
1. Settings > Domains
2. أضف الدومين
3. اتبع التعليمات

#### في مزود الدومين:
```
Type: CNAME
Name: @
Value: cname.vercel-dns.com

Type: CNAME
Name: www
Value: cname.vercel-dns.com
```

### الخطوة 5: الاختبار

```bash
# اختبار محلي
http://localhost:4002?tenant=newclient

# اختبار الإنتاج
https://newclient.com
```

### 1. قواعد البيانات المشتركة

- إذا أردت فصلهما، أنشئ قاعدة بيانات منفصلة لـ CAR X

### 2. الملفات الناقصة
- جميع المعارض تحتاج شعارات وأيقونات
- ضع الملفات في المجلدات المناسبة

### 3. الدومينات غير المربوطة
- المعارض التجريبية (alwaha, luxury, stars) لها دومينات لكن غير مربوطة
- إما ربطها أو حذفها من tenants.json

- تأكد من تغيير كلمات المرور في MongoDB
- استخدم IP Whitelist في MongoDB Atlas
- فعّل SSL/TLS للاتصالات
