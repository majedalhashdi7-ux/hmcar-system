### الخطوة 2: تحويل client-app إلى مستودع منفصل

```powershell
# 1. الانتقال إلى مجلد client-app
cd C:\car-auction\client-app

# 2. إزالة ارتباط Git الحالي (إن وجد)
Remove-Item -Recurse -Force .git -ErrorAction SilentlyContinue

# 3. تهيئة Git جديد
git init

# 4. إضافة جميع الملفات
git add .

# 5. أول commit
git commit -m "Initial commit - HM CAR Client App"

# 6. ربط بالمستودع الجديد
git remote add origin https://github.com/majedalhashdi7-ux/hmcar-client-app.git

# 7. رفع الكود
git branch -M main
git push -u origin main
```

---

### الخطوة 3: ربط client-app بـ Vercel

1. اذهب إلى: https://vercel.com/new
2. اختر: Import Git Repository
3. اختر: `hmcar-client-app`
4. Framework: Next.js
5. Root Directory: `./`
6. Environment Variables:
   ```
   NEXT_PUBLIC_API_URL=https://hmcar-system.vercel.app/api/v2
   NEXT_PUBLIC_TENANT=hmcar
   ```
7. انقر: Deploy

## 📝 ملاحظات مهمة

### 1. Environment Variables
كل نظام يحتاج متغيرات بيئة خاصة:

**Backend (.env):**
```env
MONGODB_URI=...
JWT_SECRET=...
NODE_ENV=production
```

**Client App (.env.local):**
```env
NEXT_PUBLIC_API_URL=https://hmcar-system.vercel.app/api/v2
NEXT_PUBLIC_TENANT=hmcar
```

**CarX System (.env.local):**
```env
NEXT_PUBLIC_API_URL=https://hmcar-system.vercel.app/api/v2
NEXT_PUBLIC_TENANT=carx
```

### 2. CORS Settings
تأكد من إضافة domains في Backend:
```javascript
const allowedOrigins = [
  'https://client-app-hmcar.vercel.app',
  'https://carx-system.vercel.app'
];
```

### 3. Git Ignore
تأكد من `.gitignore` في كل مستودع:
```
node_modules/
.env
.env.local
.next/
.vercel/
```
