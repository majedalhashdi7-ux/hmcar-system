# دليل النشر والتحقق من المشاكل 🚀

## المشكلة الحالية ⚠️

الموقع `hmcar.okigo.net` يعطي خطأ في الاتصال (EAI_AGAIN). هذا يعني:

### الأسباب المحتملة:
1. **DNS غير مُعد بشكل صحيح** - الدومين غير موجه إلى Vercel
2. **Vercel غير مرتبط بالدومين** - لم يتم إضافة الدومين في Vercel
3. **مشكلة في الإنترنت المحلي** - انقطاع مؤقت في الاتصال
4. **الدومين منتهي الصلاحية** - يحتاج تجديد

---

## الحل خطوة بخطوة 🔧

### الخطوة 1: تحقق من Vercel Dashboard

1. افتح: https://vercel.com/dashboard
2. سجل دخول بحسابك
3. اختر مشروع `car-auction`
4. اذهب إلى **Settings** → **Domains**

**تحقق من:**
- ✅ هل `hmcar.okigo.net` موجود في القائمة؟
- ✅ هل الحالة "Active" أم "Pending"؟
- ✅ هل هناك تحذيرات أو أخطاء؟

---

### الخطوة 2: إضافة الدومين (إذا لم يكن موجوداً)

**في Vercel Dashboard:**
1. اذهب إلى Settings → Domains
2. اضغط "Add Domain"
3. أدخل: `hmcar.okigo.net`
4. اضغط "Add"

**سيعطيك Vercel معلومات DNS:**
```
Type: A
Name: @
Value: 76.76.21.21

Type: CNAME
Name: www
Value: cname.vercel-dns.com
```

---

### الخطوة 3: إعداد DNS عند مزود الدومين

**اذهب إلى لوحة تحكم الدومين (مثل Namecheap, GoDaddy, إلخ):**

1. ابحث عن "DNS Settings" أو "DNS Management"
2. أضف السجلات التالية:

**السجل الأول (A Record):**
```
Type: A
Host: @
Value: 76.76.21.21
TTL: Automatic
```

**السجل الثاني (CNAME):**
```
Type: CNAME
Host: www
Value: cname.vercel-dns.com
TTL: Automatic
```

3. احفظ التغييرات
4. انتظر 5-30 دقيقة حتى ينتشر DNS

---

### الخطوة 4: إعادة النشر القسري

**إذا كان DNS صحيح لكن المشكلة مستمرة:**

```bash
# الطريقة 1: باستخدام السكريبت
npm run force:deploy

# الطريقة 2: يدوياً
git commit --allow-empty -m "🔄 إعادة نشر قسري"
git push origin main
```


## إذا استمرت المشكلة 🆘

### الحل البديل: استخدام رابط Vercel المباشر

1. اذهب إلى Vercel Dashboard
2. ستجد رابط مثل: `car-auction-xxx.vercel.app`
3. استخدم هذا الرابط مؤقتاً حتى يتم إصلاح الدومين

### تحقق من:

1. **حالة الدومين:**
   - هل الدومين نشط؟
   - هل انتهت صلاحيته؟
   - هل تم دفع الرسوم؟

2. **إعدادات Vercel:**
   - هل المشروع متصل بـ GitHub؟
   - هل Auto Deploy مفعل؟
   - هل هناك أخطاء في Build Logs؟

3. **إعدادات DNS:**
   - هل السجلات صحيحة؟
   - هل انتظرت وقت كافي (30 دقيقة)؟
