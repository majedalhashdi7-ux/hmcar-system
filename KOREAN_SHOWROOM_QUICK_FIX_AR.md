# حل سريع: السيارات لا تظهر في المعرض الكوري

## المشكلة
السيارات الكورية لا تظهر للعملاء في صفحة المعرض (`/showroom`)

## الحل السريع (3 خطوات)

### 1️⃣ تشخيص المشكلة
```bash
node scripts/check-korean-showroom.js
```

سيعرض لك:
- ✅ عدد السيارات الكورية الموجودة
- ⚠️  المشاكل الموجودة
- 💡 التوصيات للإصلاح

### 2️⃣ إصلاح المشكلة

#### إذا كانت المشكلة: "لا يوجد رابط Encar"
```bash
node scripts/setup-korean-showroom.js
```

#### إذا كانت المشكلة: "السيارات موجودة لكن غير مصنفة"
```bash
node scripts/fix-korean-cars.js
```

#### إذا كانت المشكلة: "لا توجد سيارات"
من لوحة الأدمن:
1. اذهب إلى: **الإعدادات** > **المعرض الكوري**
2. اضغط على: **جلب السيارات**

### 3️⃣ التحقق من الحل
افتح المتصفح واذهب إلى:
```
https://hmcar.xyz/showroom
```

يجب أن تظهر السيارات الآن! ✅

## شرح المشكلة

السيارات الكورية يجب أن تكون مصنفة بشكل صحيح في قاعدة البيانات:

```javascript
{
  source: 'korean_import',      // ✅ المصدر: كوري
  listingType: 'showroom',      // ✅ النوع: معرض
  isActive: true,               // ✅ نشطة
  isSold: false                 // ✅ غير مباعة
}
```

## الأسباب الشائعة

### ❌ السبب 1: لا توجد سيارات في قاعدة البيانات
**الحل:** اجلب سيارات من Encar (من لوحة الأدمن)

### ❌ السبب 2: السيارات موجودة لكن غير مصنفة
**الحل:** `node scripts/fix-korean-cars.js`

### ❌ السبب 3: رابط Encar غير محفوظ
**الحل:** `node scripts/setup-korean-showroom.js`

## أدوات الإصلاح

### 🔍 `check-korean-showroom.js`
يفحص حالة المعرض ويعرض التقرير الكامل

### ⚙️ `setup-korean-showroom.js`
يضيف رابط Encar الافتراضي ويفعل الإعدادات

### 🔧 `fix-korean-cars.js`
يصلح تصنيف السيارات الموجودة

## من لوحة الأدمن

### جلب سيارات جديدة
1. **الإعدادات** > **المعرض الكوري**
2. تأكد من وجود رابط Encar
3. اضغط **جلب السيارات**

### تحديث رابط Encar
1. **الإعدادات** > **المعرض الكوري**
2. الصق رابط Encar الجديد
3. احفظ التغييرات

## التحقق من النجاح

### ✅ من المتصفح
افتح: `/showroom` - يجب أن تظهر السيارات

### ✅ من API
```bash
curl https://hmcar.xyz/api/v2/showroom/cars?page=1
```

يجب أن ترجع سيارات في `data`

### ✅ من قاعدة البيانات
```javascript
db.cars.find({
  source: 'korean_import',
  isActive: true
}).count()
```

يجب أن يكون العدد > 0

1. **الجلب التلقائي:** إذا لم توجد سيارات في قاعدة البيانات، النظام يحاول جلبها مباشرة من Encar

2. **التصنيف التلقائي:** عند إضافة سيارة جديدة، النظام يصنفها تلقائيًا بناءً على البيانات

3. **الكاش:** قد تحتاج إلى تحديث الصفحة (Ctrl+F5) لرؤية التغييرات
