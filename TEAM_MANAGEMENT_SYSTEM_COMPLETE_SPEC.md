# 🏢 نظام إدارة الفريق والموقع العام - المواصفات الكاملة

## 📋 نظرة عامة على المشروع

**اسم المشروع:** نظام إدارة الفريق والموقع العام  
**النوع:** نظام ويب شامل (Next.js + Node.js + MongoDB)  
**الهدف:** نظام متكامل لإدارة الفريق البرمجي + موقع عام لعرض الخدمات  
**التاريخ:** مارس 2026  

## 🎯 الفكرة الأساسية

نظام شامل مقسم إلى قسمين رئيسيين:

### القسم الأول: لوحة الإدارة الداخلية
- إدارة العملاء والمشاريع (HMCAR, CARX, وعملاء جدد)
- إدارة الفريق والمهام اليومية
- نظام الفوترة والاشتراكات مع التنبيهات
- مراقبة النظام والأداء (uptime, response time)
- التقارير المالية والإحصائيات
- نظام أمان متقدم مع صلاحيات متعددة

### القسم الثاني: الموقع العام للعملاء
- صفحة رئيسية احترافية تعرض خدمات الشركة
- معرض أعمال يعرض HMCAR و CARX ومشاريع أخرى
- صفحة خدمات مع أسعار وباقات واضحة
- نظام طلب خدمة مع تسعير تلقائي
- نظام إدارة محتوى من لوحة الإدارة
- دعم عربي/إنجليزي مع تصميم متجاوب
- تحسين محركات البحث (SEO)
- نظام تحليلات لمتابعة الزوار

### التكامل الذكي
- طلبات الخدمة تظهر في لوحة الإدارة
- إحصائيات الموقع في التقارير الداخلية
- إدارة محتوى الموقع من لوحة الإدارة
- نظام CRM لمتابعة العملاء المحتملين

## 🛠️ التقنيات المطلوبة

### Frontend
- **Next.js 14+** (React Framework)
- **TypeScript** للأمان والجودة
- **Tailwind CSS** للتصميم
- **Framer Motion** للحركات والانتقالات
- **React Hook Form** للنماذج
- **Recharts** للرسوم البيانية

### Backend
- **Node.js + Express** للخادم
- **MongoDB + Mongoose** لقاعدة البيانات
- **JWT** للمصادقة
- **Nodemailer** للإيميلات
- **Multer** لرفع الملفات
- **Cron Jobs** للمهام المجدولة

### أدوات إضافية
- **Vercel** للاستضافة
- **MongoDB Atlas** لقاعدة البيانات
- **Cloudinary** لإدارة الصور
- **Google Analytics** للتحليلات
- **WhatsApp Business API** للتواصل

## 🔧 الميزات التفصيلية

### القسم الداخلي (لوحة الإدارة)

#### 1. إدارة العملاء والمشاريع
- **إضافة عميل جديد:** نموذج شامل مع كل التفاصيل
- **تتبع المشاريع:** حالة كل مشروع ومراحل التطوير
- **إدارة قواعد البيانات:** MongoDB URIs وتفاصيل الاتصال
- **إدارة الدومينات:** الأساسي والإضافية وحالة DNS
- **منصات الاستضافة:** Vercel, VPS, AWS مع التفاصيل

#### 2. نظام الفوترة والاشتراكات
- **تتبع الاشتراكات:** MongoDB Atlas, Vercel Pro, الدومينات
- **تنبيهات التجديد:** إشعارات قبل 7 أيام من انتهاء الاشتراك
- **حساب الإيرادات:** إجمالي الإيرادات من كل عميل
- **تقارير مالية:** شهرية وسنوية مع الرسوم البيانية
- **إدارة الفواتير:** إنشاء وإرسال الفواتير تلقائياً

#### 3. إدارة الفريق والمهام
- **توزيع المهام:** تخصيص مهام يومية وأسبوعية
- **تتبع الأداء:** ساعات العمل وجودة الكود
- **مراقبة AI:** نسبة استخدام الذكاء الاصطناعي
- **تقييم الفريق:** تقارير أداء شهرية
- **إدارة الوقت:** تتبع الوقت المستغرق في كل مشروع

#### 4. مراقبة النظام
- **فحص المواقع:** كل 5 دقائق لجميع مواقع العملاء
- **مراقبة قواعد البيانات:** حالة الاتصال والأداء
- **تنبيهات فورية:** عند حدوث مشاكل أو بطء
- **تقارير الأداء:** uptime وresponse time
- **النسخ الاحتياطية:** يومية مع التحقق من السلامة

#### 5. التقارير والإحصائيات
