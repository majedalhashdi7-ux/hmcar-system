## 🔄 تدفق الطلب (Request Flow)

```
1. المستخدم يفتح
   ┌─────────────────────┐
   │ alwaha-cars.com     │
   └──────────┬──────────┘
              │
              ▼
2. DNS يوجه إلى السيرفر
   ┌─────────────────────┐
   │ Server IP           │
   │ 76.76.21.21         │
   └──────────┬──────────┘
              │
              ▼
3. tenant-resolver يحدد المعرض
   ┌─────────────────────┐
   │ Domain: alwaha.com  │
   │ → Tenant: alwaha    │
   └──────────┬──────────┘
              │
              ▼
4. tenant-db-manager يفتح الاتصال
   ┌─────────────────────┐
   │ MongoDB Connection  │
   │ → alwaha_db         │
   └──────────┬──────────┘
              │
              ▼
5. tenantMiddleware يضيف البيانات
   ┌─────────────────────┐
   │ req.tenant = {      │
   │   name: "الواحة"    │
   │   theme: {...}      │
   │   models: {...}     │
   │ }                   │
   └──────────┬──────────┘
              │
              ▼
6. Route يعالج الطلب
   ┌─────────────────────┐
   │ const Car =         │
   │   req.getModel()    │
   │ const cars =        │
   │   await Car.find()  │
   └──────────┬──────────┘
              │
              ▼
7. Response يُرسل للمستخدم
   ┌─────────────────────┐
   │ {                   │
   │   tenant: "الواحة"  │
   │   cars: [...]       │
   │ }                   │
   └─────────────────────┘
```
