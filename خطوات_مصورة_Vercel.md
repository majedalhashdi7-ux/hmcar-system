# 📸 خطوات مصورة لإصلاح CAR X على Vercel

## 🎯 الهدف
إصلاح مشكلة Build Failed في carx-system عن طريق تصحيح Environment Variables

---

## الخطوة 1️⃣: افتح Vercel Dashboard

1. اذهب إلى: **https://vercel.com/dashboard**
2. ستجد قائمة بمشاريعك
3. ابحث عن مشروع **carx-system**
4. اضغط عليه

```
┌─────────────────────────────────────┐
│  Vercel Dashboard                   │
├─────────────────────────────────────┤
│  Projects:                          │
│  ✅ hmcar-client-app    (Ready)    │

│  ❌ carx-system         (Failed)   │ ← اضغط هنا

```

## الخطوة 2️⃣: اذهب إلى Settings

1. في صفحة المشروع، ستجد تبويبات في الأعلى
2. اضغط على تبويب **Settings**

```
┌─────────────────────────────────────┐
│  Overview  Deployments  Analytics   │
│  Settings  ← اضغط هنا               │
```

## الخطوة 3️⃣: افتح Environment Variables

1. في القائمة الجانبية اليسرى
2. اضغط على **Environment Variables**

```
┌─────────────────────────────────────┐
│  Settings                           │

│  General                            │
│  Domains                            │
│  Environment Variables  ← اضغط هنا  │

```

---

## الخطوة 4️⃣: احذف المتغير الخاطئ

1. ابحث عن متغير اسمه: `NEXT_PUBLIC_APIURL` (بدون underscore)
2. اضغط على الثلاث نقاط (⋯) بجانبه
3. اختر **Delete**
4. أكد الحذف

```
┌─────────────────────────────────────────────────┐
│  Environment Variables                          │
├─────────────────────────────────────────────────┤
│  Name                    Value              ⋯   │
│  NEXT_PUBLIC_APIURL     https://...        ⋯   │ ← احذف هذا
│                                            ↓    │
│                                         Delete  │
└─────────────────────────────────────────────────┘
```

## الخطوة 6️⃣: أعد النشر (Redeploy)

1. اضغط على تبويب **Deployments** في الأعلى
2. ستجد قائمة بجميع عمليات النشر
3. اضغط على أحدث deployment (الأول في القائمة)
4. اضغط على الثلاث نقاط (⋯) في الأعلى
5. اختر **Redeploy**
6. أكد العملية

```
┌─────────────────────────────────────────────────┐
│  Deployments                                    │
├─────────────────────────────────────────────────┤
│  Latest Deployment                          ⋯   │ ← اضغط هنا
│  Status: Failed                             ↓   │
│  Branch: main                          Redeploy │ ← اختر هذا
│  Commit: c163642                                │
└─────────────────────────────────────────────────┘
```

## ⚠️ نصائح مهمة

1. **انتبه للأسماء**
   - `NEXT_PUBLIC_API_URL` (مع underscore بين API و URL)
   - ليس `NEXT_PUBLIC_APIURL` (بدون underscore)

2. **اختر جميع Environments**
   - Production ✅
   - Preview ✅
   - Development ✅

3. **لا تنسى Redeploy**
   - المتغيرات لا تطبق تلقائياً
   - يجب إعادة النشر

4. **انتظر البناء**
   - لا تغلق الصفحة
   - انتظر حتى ترى ✅ Ready
