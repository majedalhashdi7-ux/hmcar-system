## 🔥 ما تبقى (3 خطوات فقط):

### الخطوة 1: إنشاء قواعد البيانات (5 دقائق)

1. **اذهب إلى**: https://cloud.mongodb.com
2. **أنشئ حساب** أو سجل دخول
3. **أنشئ Cluster جديد** (مجاني)
4. **أنشئ قاعدتي بيانات**:
   - `hmcar_production` (لـ HM CAR)
   - `carx_production` (لـ CAR X)
5. **احصل على Connection Strings**

### الخطوة 2: إعداد Vercel (5 دقائق)

1. **اذهب إلى**: https://vercel.com
2. **أنشئ مشروعين**:
   - مشروع 1: ربط مجلد `client-app` (HM CAR)
   - مشروع 2: ربط مجلد `carx-system` (CAR X)
3. **أضف متغيرات البيئة** لكل مشروع:
   ```
   MONGO_URI=mongodb+srv://username:password@cluster.mongodb.net/hmcar_production
   MONGO_URI_CARX=mongodb+srv://username:password@cluster.mongodb.net/carx_production
   NEXTAUTH_SECRET=your-secret-key-here
   NEXTAUTH_URL=https://your-domain.vercel.app
   ADMIN_PASSWORD=your-admin-password
   ```

### الخطوة 3: النشر (2 دقيقة)

```bash
# نشر HM CAR
cd client-app
vercel --prod

# نشر CAR X  
cd ../carx-system
vercel --prod
```
