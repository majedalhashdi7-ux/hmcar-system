```javascript
// في tenant-resolver.js

- ?tenant=carx

الطريقة 4: Default
- defaultTenant: "hmcar"
```

### 2. اتصال قاعدة البيانات
```javascript
// في tenant-db-manager.js

كل عميل له:
- اتصال MongoDB منفصل

- بيانات معزولة تماماً

مثال:
hmcar → mongodb://...hmcar_production
carx  → mongodb://...carx_production
```

### 3. API Requests
```javascript
// في tenantMiddleware.js

كل طلب API:
1. يحدد العميل من الطلب
2. يتصل بقاعدة بيانات العميل
3. يخزن معلومات العميل في req.tenant
4. يوفر Models في req.tenantModels

مثال:
GET /api/v2/cars
Host: daood.okigo.net
→ يجلب سيارات hmcar فقط

GET /api/v2/cars
Host: carx-system.vercel.app
→ يجلب سيارات carx فقط
```

### 4. Frontend (TenantContext)
```typescript
// في client-app/src/lib/TenantContext.tsx

عند تحميل الصفحة:
1. يستدعي /api/v2/tenant/info

3. يطبق الثيم على الصفحة
4. يحدث favicon و title

مثال:
hmcar → ذهبي (#D4AF37)
```

```json
{
  "routes": [
    {
      "src": "/api/(.*)",
      "dest": "vercel-server.js"  // ✅ صحيح
    },
    {
      "src": "/(.*)",
      "dest": "client-app/$1"  // ❌ خطأ - يعرض HM CAR فقط

  ]
```

**النتيجة:**
- daood.okigo.net/ → يعرض client-app (HM CAR)

**المشكلة:**
- client-app مرتبط بـ HM CAR فقط
