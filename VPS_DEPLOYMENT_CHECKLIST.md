- [ ] `BASE_URL` محدث بالدومين الجديد
- [ ] `ALLOWED_ORIGINS` محدث بالدومين الجديد
- [ ] `MONGO_URI` صحيح
- [ ] `REDIS_URL` صحيح
- [ ] `CLOUDINARY_*` صحيح
- [ ] `EMAIL_*` صحيح
- [ ] `JWT_SECRET` موجود
- [ ] `SESSION_SECRET` موجود
- [ ] `NODE_ENV=production`
- [ ] `PORT=4001`

### Frontend (client-app/.env.production):
- [ ] `NEXT_PUBLIC_API_URL` محدث
- [ ] `NEXT_PUBLIC_SOCKET_URL` محدث
- [ ] `NODE_ENV=production`
