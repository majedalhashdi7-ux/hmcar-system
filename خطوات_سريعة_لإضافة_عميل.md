# ⚡ خطوات سريعة لإضافة عميل جديد

## 🎯 الطريقة السريعة (5 دقائق)

### 1️⃣ شغّل السكريبت
```bash
node scripts/add-tenant.js
```

### 2️⃣ أدخل البيانات
```
معرف المعرض: newclient
اسم المعرض (عربي): معرض العميل الجديد
اسم المعرض (إنجليزي): New Client Motors
الدومين: newclient.com
MongoDB URI: mongodb+srv://user:pass@cluster.mongodb.net/newclient_db
واتساب: +966501234567
بريد: info@newclient.com
اللون: #3498db
```

### 3️⃣ ارفع الملفات
```bash
# ضع الشعار
uploads/tenants/newclient/logo.png

# ضع الأيقونة
uploads/tenants/newclient/favicon.ico
```

### 4️⃣ اربط الدومين في Vercel
1. اذهب إلى https://vercel.com/dashboard
2. Settings > Domains
3. أضف: newclient.com
4. اتبع التعليمات

### 5️⃣ اختبر
```bash
# محلياً
http://localhost:4002?tenant=newclient

# على الإنتاج
https://newclient.com
```
