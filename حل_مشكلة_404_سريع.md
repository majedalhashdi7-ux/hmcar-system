# 🔧 حل مشكلة 404 - خطوات سريعة

**المشكلة:** خطأ 404 عند فتح https://hmcar-client-app.vercel.app  
**السبب:** Environment Variables غير موجودة على Vercel  
**الحل:** 3 خطوات سريعة (5 دقائق)

---

## ✅ الخطوة 1: افتح إعدادات Vercel

1. اذهب إلى: https://vercel.com/dashboard
2. اختر المشروع: **hmcar-client-app**
3. اضغط على: **Settings** (في الأعلى)
4. من القائمة الجانبية، اختر: **Environment Variables**

---

## ✅ الخطوة 2: أضف المتغيرات (انسخ والصق)

**أضف هذه المتغيرات واحدة تلو الأخرى:**

### المتغير 1:
```
Name:  NEXT_PUBLIC_API_URL
Value: https://hmcar-system.vercel.app/api/v2
```
- اختر: **Production**, **Preview**, **Development** (الثلاثة)
- اضغط: **Save**

### المتغير 2:
```
Name:  NEXT_PUBLIC_TENANT
Value: hmcar
```
- اختر: **Production**, **Preview**, **Development** (الثلاثة)
- اضغط: **Save**

### المتغير 3:
```
Name:  NEXT_PUBLIC_APP_NAME
Value: HM CAR
```
- اختر: **Production**, **Preview**, **Development** (الثلاثة)
- اضغط: **Save**

### المتغير 4:
```
Name:  NEXT_PUBLIC_WHATSAPP
Value: +967781007805
```
- اختر: **Production**, **Preview**, **Development** (الثلاثة)
- اضغط: **Save**

### المتغير 5:
```
Name:  NODE_ENV
Value: production
```
- اختر: **Production** فقط
- اضغط: **Save**

---

## ✅ الخطوة 3: إعادة النشر (Redeploy)

**الطريقة الأسهل:**

1. اذهب إلى: **Deployments** (في الأعلى)
2. اختر آخر deployment (الأول في القائمة)
3. اضغط على الثلاث نقاط **...** على اليمين
4. اختر: **Redeploy**
5. اضغط: **Redeploy** مرة أخرى للتأكيد
6. انتظر 2-3 دقائق حتى ينتهي النشر

## 🔍 إذا استمرت المشكلة

### تحقق من Logs:

1. في Vercel Dashboard
2. اذهب إلى: **Deployments**
3. اختر آخر deployment
4. اضغط على: **View Function Logs**
5. ابحث عن أي أخطاء

### تحقق من Build:

1. في Vercel Dashboard
2. اذهب إلى: **Deployments**
3. اختر آخر deployment
4. اضغط على: **Building**
5. تأكد من أن Build نجح بدون أخطاء
