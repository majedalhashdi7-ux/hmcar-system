## 🚀 إضافة عميل جديد (3 خطوات)

### 1. شغّل السكريبت
```bash
node scripts/add-tenant.js
```

### 2. ارفع الملفات
```bash
uploads/tenants/[id]/logo.png
uploads/tenants/[id]/favicon.ico
```

### 3. اربط الدومين
- اذهب إلى Vercel Dashboard
- Settings > Domains
- أضف الدومين

---

## 🔍 فحص النظام

```bash
node scripts/check-system.js
```
