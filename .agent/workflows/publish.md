هذا التدفق يساعدك على حفظ تعديلاتك، رفعها إلى GitHub، ونشرها على Vercel لتظهر على الموقع.

## خطوات النشر السريع (Quick Publish)

// turbo

1. تنفيذ الأوامر التالية بالترتيب:
   - إضافة الملفات: `git add .`
   - عمل Commit: `git commit -m "update: latest changes and improvements"`
   - رفع الكود: `git push`

2. النشر على Vercel لضمان التحديث الفوري (Production):
// turbo
   - تشغيل أمر النشر: `vercel --prod`

## ملاحظات هامة

- بعد تنفيذ `git push` قد يستغرق Vercel من 1-3 دقائق لبناء النسخة الجديدة.
- إذا كنت تستخدم `vercel --prod` سيتم النشر بشكل أسرع ومباشر.
- لرؤية التغييرات فوراً في المتصفح، استخدم **Ctrl + F5** لتجاوز الذاكرة المؤقتة (Cache).
