## 🚀 خطوات إضافة عميل جديد

### الطريقة 1: استخدام السكريبت التلقائي (الأسهل) ⭐

```bash
# تشغيل السكريبت
node scripts/add-tenant.js
```

سيطلب منك:
1. معرف المعرض (بالإنجليزي، مثل: newclient)
2. اسم المعرض بالعربي
3. اسم المعرض بالإنجليزي
4. الدومين (مثال: newclient.com)
5. MongoDB URI
6. رقم الواتساب
7. البريد الإلكتروني
8. اللون الأساسي

**مثال:**
```
1️⃣  معرف المعرض: newclient
2️⃣  اسم المعرض (بالعربي): معرض العميل الجديد
3️⃣  اسم المعرض (بالإنجليزي): New Client Motors
4️⃣  الدومين: newclient.com
5️⃣  MongoDB URI: mongodb+srv://user:pass@cluster.mongodb.net/newclient_db
6️⃣  رقم الواتساب: +966501234567
7️⃣  البريد الإلكتروني: info@newclient.com
8️⃣  اللون الأساسي: #3498db
```

```json
{

  "tenants": {
    "hmcar": { ... },
    "carx": { ... },

      "nameEn": "New Client Motors",
      "description": "معرض العميل الجديد للسيارات",
      "descriptionEn": "New Client Motors Showroom",
      "domains": [
        "newclient.com",
        "www.newclient.com",
        "newclient-system.vercel.app"
      ],
      "mongoUri": "mongodb+srv://user:password@cluster.mongodb.net/newclient_db?retryWrites=true&w=majority",
      "logo": "/uploads/tenants/newclient/logo.png",
      "favicon": "/uploads/tenants/newclient/favicon.ico",
      "theme": {
        "primaryColor": "#3498db",
        "secondaryColor": "#2980b9",
        "accentColor": "#e74c3c",
        "backgroundColor": "#ffffff",
        "textColor": "#2c3e50"
      },

        "whatsapp": "+966501234567",
        "email": "info@newclient.com",
        "phone": "+966501234567"
      },
      "settings": {
        "currency": "SAR",
        "language": "ar",
        "direction": "rtl"
      },
      "enabled": true
    }
  }

```

#### الخطوة 2: إنشاء قاعدة بيانات MongoDB

1. اذهب إلى [MongoDB Atlas](https://cloud.mongodb.com)
2. اضغط "Create Database"
3. اسم Database: `newclient_db`
4. انسخ Connection String
5. ضعه في `mongoUri` في الملف أعلاه

**أو استخدم نفس قاعدة البيانات:**
```json
"mongoUri": "ENV:MONGO_URI"
```

```bash
# إنشاء المجلد
mkdir -p uploads/tenants/newclient

# رفع الملفات
```

## 🌐 ربط الدومين

### الخيار 1: ربط دومين خارجي (GoDaddy, Namecheap, إلخ)

#### أ. في لوحة تحكم الدومين:

**إذا كان لديك VPS/Server:**
```
Type: A
Name: @
Value: [IP السيرفر]
TTL: 3600

Type: A
Name: www
Value: [IP السيرفر]
TTL: 3600
```

**إذا كنت تستخدم Vercel:**
```
Type: CNAME
Name: @
Value: cname.vercel-dns.com
TTL: 3600

Type: CNAME
Name: www
Value: cname.vercel-dns.com
TTL: 3600
```

#### ب. في Vercel Dashboard:

1. اذهب إلى [Vercel Dashboard](https://vercel.com/dashboard)
2. اختر المشروع
3. Settings > Domains
4. أضف الدومين: `newclient.com`
5. أضف أيضاً: `www.newclient.com`
6. اتبع التعليمات للتحقق

---

### الخيار 2: استخدام Subdomain من Vercel (للاختبار)

الدومين يُنشأ تلقائياً:
```
https://[project-name]-[hash].vercel.app
```

أضفه في `domains` في `tenants.json`:
```json
"domains": [
  "newclient-system.vercel.app",
  "newclient.com"
]
```

## 🧪 اختبار العميل الجديد

### 1. اختبار محلي (بدون دومين)

```bash
# عبر Query Parameter
http://localhost:4002?tenant=newclient

# عبر Header (للـ API)
curl -H "X-Tenant-ID: newclient" http://localhost:4002/api/v2/tenant/info
```

### 2. اختبار على الإنتاج

```bash
# عبر الدومين
https://newclient.com

# عبر Query Parameter (للاختبار)
https://hmcar.xyz?tenant=newclient

# فحص API
curl https://newclient.com/api/v2/tenant/info
```

### 3. التحقق من قاعدة البيانات

```bash
# سكريبت فحص الاتصال
node -e "
const mongoose = require('mongoose');
mongoose.connect('mongodb+srv://...').then(() => {
  console.log('✅ الاتصال نجح');
  process.exit(0);
}).catch(err => {
  console.error('❌ فشل الاتصال:', err.message);
  process.exit(1);
});
"
```

```bash
# إنشاء سكريبت الفحص

const { getAllTenants } = require('../tenants/tenant-resolver');

const tenants = getAllTenants();
console.log(`✅ عدد المعارض المفعّلة: ${tenants.length}\n`);

  console.log(`   اللون: ${t.theme.primaryColor}\n`);

const connections = getConnectionsStatus();
connections.forEach(c => {
```

## 💡 نصائح مهمة

### 1. استخدام نفس قاعدة البيانات

إذا أردت استخدام نفس قاعدة البيانات لعدة عملاء:
```json
"mongoUri": "ENV:MONGO_URI"
```

**ملاحظة:** البيانات ستكون مشتركة بين العملاء!

### 2. قواعد بيانات منفصلة (موصى به)

لكل عميل قاعدة بيانات خاصة:
```json
"mongoUri": "mongodb+srv://user:pass@cluster.mongodb.net/client1_db"
```

### 3. استخدام متغيرات البيئة

في `.env`:
```bash
MONGO_URI=mongodb+srv://...
NEWCLIENT_MONGO_URI=mongodb+srv://...
```

في `tenants.json`:
```json
"mongoUri": "ENV:NEWCLIENT_MONGO_URI"
```

### 4. تعطيل عميل مؤقتاً

```json
{
  "enabled": false
}
```
