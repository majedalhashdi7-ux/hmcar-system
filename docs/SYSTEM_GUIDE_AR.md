# دليل نظام HM CAR - التوثيق الشامل

أهلاً بك في التوثيق الرسمي لنظام **HM CAR**، هذا النظام متكامل لإدارة مزادات السيارات وبيع قطع الغيار، تم بناؤه بأحدث التقنيات لضمان السرعة، الأمان، والتجربة المستخدم السينمائية.

---

## 🏗 هيكلية النظام (Architecture)

النظام مقسم إلى جزأين رئيسيين:

### 1. الواجهة الأمامية (Frontend - client-app)

- **الإطار البرمجي:** Next.js (App Router).
- **التصميم:** Tailwind CSS مع Vanilla CSS للتخصيص الدقيق.
- **الحركات:** Framer Motion لخلق تجربة سينمائية سلسة.
- **المكونات الرئيسية:**
  - `CinematicVideoBackground`: المسؤول عن خلفية الفيديو الديناميكية.
  - `LandingShowcase`: عرض السيارات المميزة بطريقة جذابة.
  - `Navbar` & `AdminHeader`: أنظمة التنقل للمستخدم والمدير.

### 2. الواجهة الخلفية (Backend)

- **البيئة:** Node.js مع Express.
- **قاعدة البيانات:** MongoDB (عبر Mongoose).
- **إدارة الحالة:** WebSockets (Socket.io) للتحديثات الفورية للمزادات.
- **الخدمات (Services):**
  - `PaymentService`: معالجة المدفوعات.
  - `NotificationService`: إرسال التنبيهات المتقدمة.
  - `InventoryService`: إدارة المخزون والسيارات.
  - `SEOService`: تحسين محركات البحث تلقائياً.

---

## 📁 تنظيم الملفات (File Organization - الهيلة)

تم تنظيم المشروع ليكون واضحاً وسهل الصيانة:

- **`/models`**: تحتوي على تعريفات الجداول (Schemas) مثل السيارات، المزادات، المستخدمين، والطلبات.
- **`/routes`**: تحتوي على نقاط الاتصال (API Endpoints).
- **`/services`**: تحتوي على المنطق البرمجي المعقد (Business Logic) لضمان فصل المهام.
- **`/middleware`**: طبقات الحماية والتحقق من الصلاحيات.
- **`/client-app/src`**: الكود المصدري للواجهة الأمامية.

---

## 🛠 المميزات التقنية

1. **الخلفية الذكية:** يتم التبديل تلقائياً بين الفيديو (على أجهزة الكمبيوتر) والصورة (على الجوال) لضمان أفضل أداء.
2. **نظام المزادات المباشر:** تحديثات فورية للأسعار والمزايدات عبر WebSockets.
3. **لوحة تحكم الأدمن:** شاملة لإدارة كل شيء من إعدادات الموقع إلى تقارير المبيعات بنظام النسخ الاحتياطي (Backup).
4. **الأمان:** تشفير البيانات، حماية 2FA، ونظام أدوار (Roles) متقدم.

---

## 🚀 كيفية التشغيل والتطوير

1. **تثبيت المكتبات:** `npm install` في المجلد الرئيسي وفي مجلد `client-app`.
2. **إعدادات البيئة:** تأكد من ضبط ملف `.env` لربط قاعدة البيانات والمفاتيح السرية.
3. **تشغيل النظام:**
    - للأدمن/الخلفية: `npm start` أو `node server.js`.
    - للواجهة الأمامية: `cd client-app && npm run dev`.
